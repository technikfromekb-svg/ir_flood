# Flipper Zero IR Multi-Flood Pro 🚀

Приложение для **Flipper Zero**, предназначенное для автоматического перебора (брутфорса) инфракрасных кодов (команд и адресов). Программа использует встроенный аппаратный ИК-передатчик и оптимизированные таймеры Furi OS для высокоскоростной отправки пакетов без зависаний интерфейса.

## 📌 Особенности
*   **Автоматический перебор:** Циклически перебирает команды от `0x00` до `0xFF`. При завершении круга команд автоматически инкрементирует адрес устройства (`0x00`-`0xFF`).
*   **Поддержка популярных протоколов:** 
    *   NEC (38kHz)
    *   SAMSUNG (38kHz)
    *   SONY 12-bit (40kHz)
    *   Philips RC5 (36kHz)
    *   Philips RC6 (36kHz)
*   **Асинхронная отправка:** Использование прерываний (ISR) гарантирует отправку полноценных ИК-пакетов без искажений таймингов.
*   **Настройка скорости:** Динамическое изменение интервала между отправками прямо во время работы устройства.
*   **Энергоэффективность:** Основной цикл приложения засыпает (`FuriWaitForever`) и просыпается только по системным прерываниям от кнопок или аппаратного таймера.

## 🎮 Управление (UI Controls)

Приложение имеет интуитивный интерфейс с выводом текущего прогресса в процентах.

*   `OK` (Короткое нажатие) — Старт / Стоп флуда (перебора).
*   `Вверх` / `Вниз` — Смена ИК-протокола (доступно только в режиме `IDLE`).
*   `Влево` / `Вправо` — Изменение интервала отправки (скорости флуда) с шагом в 10 мс (от 50 мс до 1000 мс).
*   `Назад (Back)` — Выход из приложения с корректным освобождением всех ресурсов и остановкой передатчика.

## 🛠 Требования для сборки
Для компиляции приложения вам понадобятся:
1. Установленный **Git**.
2. Официальный инструмент сборки приложений флиппера **ufbt** (микро-SDK).

---

## 🚀 Инструкция по сборке и запуску

### 1. Склонируйте репозиторий
Откройте терминал на компьютере и выполните команду:
```bash
git clone https://github.com
cd ir_multi_flood_pro
```

### 2. Подключите Flipper Zero
Подключите ваш Flipper Zero к компьютеру через USB-кабель.

### 3. Компиляция и автоматический запуск
Запустите утилиту сборки `ufbt` с флагом `launch`. Она сама скомпилирует исходный код, загрузит приложение в память Flipper Zero по USB и запустит его на экране устройства:
```bash
ufbt launch
```

### 4. Сборка файла приложения (.fap)
Если вы хотите просто собрать исполняемый файл приложения для ручной установки на SD-карту (в папку `apps/Tools/` или `apps/Plugins/`), выполните:
```bash
ufbt
```
Готовый файл `.fap` появится в директории `.fap/`.
